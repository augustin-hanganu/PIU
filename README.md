# Game Marketplace (C#)

## Descriere

Game Marketplace este o aplicație dezvoltată în limbajul **C#** care simulează o platformă de distribuție digitală pentru jocuri video, inspirată de platforme precum Steam. Proiectul permite utilizatorilor să vizualizeze jocuri disponibile, să le adauge într-o bibliotecă personală și să gestioneze colecția lor de jocuri.

Scopul acestui proiect este de a demonstra utilizarea conceptelor de programare în C#, precum programarea orientată pe obiect, gestionarea datelor și dezvoltarea unei interfețe simple pentru utilizator.

## Funcționalități

* Vizualizarea unei liste de jocuri disponibile
* Afișarea detaliilor despre un joc (titlu, preț, gen, descriere)
* Simularea cumpărării unui joc
* Gestionarea bibliotecii utilizatorului

## Scop educațional

Acest proiect a fost realizat în scop educațional pentru a exersa concepte fundamentale din C#, precum:

* clase și obiecte
* încapsulare
* colecții
* organizarea unui proiect software
